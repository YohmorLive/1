## 🚀 التثبيت والتشغيل

### 1. تحميل الملفات

انسخ جميع الملفات إلى مجلد على جهازك:
```
yohmor-live/
├── index.html
├── admin.html
├── styles.css
├── app.js
├── admin.js
├── data.json
└── README.md
```

### 2. فتح الموقع محلياً

**الطريقة الأولى - المتصفح المباشر:**
1. انقر بزر الفأرة الأيمن على ملف `index.html`
2. اختر "فتح مع" ثم اختر متصفحك المفضل

**الطريقة الثانية - خادم محلي (موصى به):**

إذا كان لديك Python مثبت:
```bash
# Python 3
python -m http.server 8000

# أو Python 2
python -m SimpleHTTPServer 8000
```

ثم افتح المتصفح وانتقل إلى: `http://localhost:8000`

إذا كان لديك Node.js:
```bash
npx http-server
```

### 3. اختبار الموقع

- افتح `index.html` لرؤية الصفحة الرئيسية
- انقر على زر ⚙️ في الزاوية العلوية اليسرى للدخول إلى لوحة الإدارة
- جرب البحث والتصفية
- أضف عمل جديد من لوحة الإدارة

## 📤 النشر على GitHub Pages

### الخطوة 1: إنشاء حساب GitHub
1. انتقل إلى [github.com](https://github.com)
2. انقر على "Sign up" وأنشئ حساباً جديداً
3. أكمل عملية التحقق

### الخطوة 2: إنشاء مستودع جديد
1. انقر على "+" في الزاوية العلوية اليمنى
2. اختر "New repository"
3. سمِّ المستودع: `yohmor-live` (أو أي اسم تفضله)
4. اختر "Public"
5. انقر على "Create repository"

### الخطوة 3: رفع الملفات
**باستخدام Git (الطريقة الموصى بها):**

```bash
# انسخ رابط المستودع من GitHub

# في مجلد المشروع:
git init
git add .
git commit -m "Initial commit - Yohmor Live"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/yohmor-live.git
git push -u origin main
```

**بدون Git (رفع مباشر):**
1. في صفحة المستودع، انقر على "Add file" > "Upload files"
2. اسحب جميع الملفات إلى منطقة التحميل
3. انقر على "Commit changes"

### الخطوة 4: تفعيل GitHub Pages
1. انتقل إلى إعدادات المستودع (Settings)
2. اختر "Pages" من القائمة الجانبية
3. تحت "Source"، اختر "main" و "root"
4. انقر على "Save"

### الخطوة 5: الوصول إلى الموقع
بعد دقيقة أو دقيقتين، سيكون موقعك متاحاً على:
```
https://YOUR_USERNAME.github.io/yohmor-live
```

## 📖 استخدام الموقع

### الصفحة الرئيسية

#### البحث
- اكتب في مربع البحث للبحث عن أعمال معينة
- البحث يعمل على الاسم والوصف والوسوم

#### التصفية
- انقر على أزرار الفئات لتصفية الأعمال
- الخيارات: الكل، مطعم، خدمات، متجر، جمعية

#### الترتيب
- اختر من قائمة الترتيب:
  - الأحدث: أحدث الأعمال أولاً
  - اسم العمل: ترتيب أبجدي

#### عرض التفاصيل
- انقر على بطاقة العمل لرؤية التفاصيل الكاملة
- يمكنك رؤية الصور والعنوان وساعات العمل
- انقر على زر "تواصل عبر واتساب" للتواصل المباشر

#### الإشعارات
- انقر على زر 🔔 الإشعارات لرؤية آخر إشعار
- الإشعارات تظهر تلقائياً عند تحميل الصفحة

### لوحة الإدارة

#### الدخول
- انقر على ⚙️ في الزاوية العلوية اليسرى
- لا توجد كلمة مرور (الموقع ثابت على GitHub Pages)

#### إدارة الأعمال
1. انتقل إلى تبويب "إدارة الأعمال"
2. ملء النموذج بمعلومات العمل
3. انقر على "حفظ العمل"
4. سيظهر العمل في القائمة أدناه

#### تعديل عمل
1. انقر على زر "تعديل" بجانب العمل
2. عدّل المعلومات
3. انقر على "حفظ العمل"

#### حذف عمل
1. انقر على زر "حذف" بجانب العمل
2. أكد الحذف

#### إدارة الإشعارات
1. انتقل إلى تبويب "إدارة الإشعارات"
2. ملء النموذج بمعلومات الإشعار
3. اختر التاريخ والوقت
4. اختياري: ثبّت الإشعار ليظهر أولاً
5. انقر على "حفظ الإشعار"

#### استيراد البيانات
1. انتقل إلى تبويب "استيراد/تصدير"
2. انقر على "استيراد"
3. اختر ملف JSON
4. سيتم تحميل جميع البيانات

#### تصدير البيانات
1. انتقل إلى تبويب "استيراد/تصدير"
2. انقر على "تصدير البيانات"
3. سيتم تحميل ملف JSON بجميع بياناتك

#### الإعدادات
1. انتقل إلى تبويب "الإعدادات"
2. عدّل معلومات الموقع والمطور
3. انقر على "حفظ الإعدادات"

## 🎨 التخصيص

### تغيير الألوان
في ملف `styles.css`، ابحث عن:
```css
:root {
    --primary-color: #0B8A4B;
    --primary-dark: #066B3A;
    --primary-light: #1E9F5D;
    /* ... */
}
```

غيّر القيم السادسة عشرية للألوان التي تفضلها.

### تغيير الخط
في `styles.css`، ابحث عن:
```css
body {
    font-family: 'Noto Kufi Arabic', 'Cairo', 'Almarai', sans-serif;
}
```

### تغيير معلومات المطور
في `admin.html`، انتقل إلى تبويب "الإعدادات" وعدّل:
- اسم المطور
- رقم واتساب
- رابط إنستغرام
