# ⚡ دليل البدء السريع - Yohmor Live

ابدأ مع Yohmor Live في 5 دقائق فقط!

## 🚀 الخطوة 1: فتح الموقع (1 دقيقة)

### الطريقة الأسهل:
1. انقر بزر الفأرة الأيمن على ملف `index.html`
2. اختر "فتح مع" ثم اختر متصفحك
3. ✅ تم! الموقع يعمل الآن

## 📝 الخطوة 2: اختبار الميزات (2 دقيقة)

### جرب البحث:
- اكتب "مطعم" في مربع البحث
- ستظهر النتائج فوراً

### جرب التصفية:
- انقر على أزرار الفئات (مطعم، خدمات، متجر، جمعية)
- الأعمال تتغير حسب الفئة المختارة

### جرب الإشعارات:
- انقر على زر 🔔 الإشعارات
- سيظهر آخر إشعار

### جرب التفاصيل:
- انقر على أي بطاقة عمل
- ستظهر نافذة بالتفاصيل الكاملة
- انقر على "تواصل عبر واتساب"

## ⚙️ الخطوة 3: الدخول إلى الإدارة (1 دقيقة)

### الدخول:
1. انقر على ⚙️ في الزاوية العلوية اليسرى
2. ستفتح لوحة الإدارة

### إضافة عمل جديد:
1. في تبويب "إدارة الأعمال"
2. ملء النموذج:
   - اسم العمل
   - الفئة
   - الوصف
   - رقم واتساب
3. انقر على "حفظ العمل"
4. ✅ تم! العمل يظهر في الصفحة الرئيسية

### إضافة إشعار:
1. انتقل إلى تبويب "إدارة الإشعارات"
2. ملء النموذج:
   - العنوان
   - الرسالة
   - التاريخ (اختياري)
3. انقر على "حفظ الإشعار"
4. ✅ تم! الإشعار يظهر في الصفحة الرئيسية

## 💾 الخطوة 4: حفظ البيانات (1 دقيقة)

### تصدير البيانات:
1. انتقل إلى تبويب "استيراد/تصدير"
2. انقر على "تصدير البيانات"
3. سيتم تحميل ملف JSON بجميع بياناتك
4. احفظ هذا الملف في مكان آمن

### استيراد البيانات:
1. انتقل إلى تبويب "استيراد/تصدير"
2. انقر على "استيراد"
3. اختر ملف JSON
4. ✅ تم! جميع البيانات محملة

## 🌐 النشر على الإنترنت (5 دقائق إضافية)

### استخدام GitHub Pages (مجاني):

#### الخطوة 1: إنشاء حساب GitHub
1. انتقل إلى [github.com](https://github.com)
2. انقر على "Sign Up"
3. أنشئ حساباً باستخدام بريدك الإلكتروني
4. تحقق من بريدك وأكمل التسجيل

#### الخطوة 2: إنشاء مستودع جديد
1. بعد تسجيل الدخول، انقر على "+" في الزاوية العلوية اليسرى
2. اختر "New repository"
3. ملء البيانات:
   - **Repository name:** `yohmor-live`
   - **Description:** منصة يحمر لايف (اختياري)
   - **Visibility:** اختر "Public"
4. انقر "Create repository"

#### الخطوة 3: رفع الملفات
1. في صفحة المستودع الجديد، انقر على "Add file" > "Upload files"
2. اسحب جميع ملفات المشروع (index.html, admin.html, styles.css, app.js, admin.js, data.json, about.html)
3. في الأسفل، انقر "Commit changes"

#### الخطوة 4: تفعيل GitHub Pages
1. انتقل إلى **Settings** (الإعدادات)
2. من القائمة اليسرى، اختر **Pages**
3. تحت "Source"، اختر:
   - Branch: `main`
   - Folder: `/ (root)`
4. انقر **Save**
5. انتظر دقيقة واحدة

#### الخطوة 5: الوصول إلى موقعك
- سيظهر رابطك: `https://YOUR_USERNAME.github.io/yohmor-live`
- شارك هذا الرابط مع الجميع!

## ✏️ تعديل المحتوى بعد النشر

### كيفية تعديل البيانات على الموقع المنشور:

#### الطريقة 1: تعديل محلي ثم رفع (الأفضل)

##### الخيار أ: استخدام Git (الأسرع والأفضل)

1. **تثبيت Git:**
   - انتقل إلى [git-scm.com](https://git-scm.com)
   - حمّل وثبّت Git
   - أعد تشغيل جهازك

2. **فتح Command Prompt/PowerShell:**
   - اضغط على `Windows + R`
   - اكتب `cmd` أو `powershell`
   - اضغط Enter

3. **الذهاب إلى مجلد المشروع:**
   ```
   cd C:\Users\dell\Desktop\1233
   ```

4. **إعداد Git (المرة الأولى فقط):**
   ```
   git config --global user.name "اسمك"
   git config --global user.email "بريدك@gmail.com"
   ```

5. **استنساخ المستودع:**
   ```
   git clone https://github.com/YOUR_USERNAME/yohmor-live.git
   ```

6. **نسخ ملفات المشروع:**
   - انسخ جميع ملفات المشروع إلى المجلد الذي تم إنشاؤه

7. **رفع التحديثات:**
   ```
   cd yohmor-live
   git add .
   git commit -m "تحديث البيانات والمحتوى"
   git push
   ```

8. **انتظر التحديث:**
   - GitHub Pages سيحدّث الموقع تلقائياً خلال دقيقة

##### الخيار ب: استخدام GitHub Desktop (الأسهل بدون سطر أوامر)

1. **تحميل GitHub Desktop:**
   - انتقل إلى [desktop.github.com](https://desktop.github.com)
   - حمّل وثبّت البرنامج

2. **تسجيل الدخول:**
   - افتح GitHub Desktop
   - انقر على "Sign in"
   - استخدم حساب GitHub الخاص بك

3. **استنساخ المستودع:**
   - انقر على "File" > "Clone repository"
   - اختر `yohmor-live`
   - اختر مجلد التخزين
   - انقر "Clone"

4. **تحديث الملفات:**
   - انسخ ملفات المشروع المحدثة إلى المجلد

5. **رفع التحديثات:**
   - في GitHub Desktop، ستظهر الملفات المتغيرة
   - اكتب رسالة في "Summary": "تحديث البيانات"
   - انقر "Commit to main"
   - انقر "Push origin"

6. **انتظر التحديث:**
   - الموقع سيتحدث خلال دقيقة

⚠️ **ملاحظة:** هذه الطريقة بطيئة جداً إذا كان لديك عدة ملفات

#### الطريقة 2: تعديل مباشر على الموقع المنشور
1. **افتح الموقع المنشور:**
   - انتقل إلى: `https://YOUR_USERNAME.github.io/yohmor-live`

2. **ادخل لوحة الإدارة:**
   - انقر على ⚙️ أو افتح: `https://YOUR_USERNAME.github.io/yohmor-live/admin.html`

3. **أضف أو عدّل البيانات:**
   - أضف أعمال جديدة
   - أضف إشعارات
   - عدّل الإعدادات

4. **احفظ البيانات:**
   - صدّر البيانات كملف JSON
   - احفظ الملف في مكان آمن

⚠️ **ملاحظة مهمة:**
- التعديلات على الموقع المنشور تُحفظ محلياً في متصفحك فقط
- إذا حذفت بيانات المتصفح، ستفقد التعديلات
- لحفظ التعديلات بشكل دائم، صدّر البيانات وأعد رفعها إلى GitHub

#### الطريقة 3: استيراج بيانات محفوظة
1. افتح لوحة الإدارة
2. انتقل إلى تبويب "استيراد/تصدير"

4. اختر ملف JSON محفوظ
5. جميع البيانات ستُحمّل فوراً

## 🎨 التخصيص السريع

### تغيير اللون الأساسي:
1. افتح `styles.css`
2. ابحث عن `--primary-color: #0B8A4B;`
3. غيّر الكود اللوني إلى اللون الذي تريده
4. احفظ الملف

### تغيير معلومات المطور:
1. افتح `admin.html`
2. انتقل إلى تبويب "الإعدادات"
3. غيّر:
   - اسم المطور
   - رقم واتساب
   - رابط إنستغرام
4. انقر "حفظ الإعدادات"
