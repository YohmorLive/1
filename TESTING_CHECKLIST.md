# ✅ قائمة اختبار Yohmor Live

استخدم هذه القائمة للتأكد من أن جميع ميزات الموقع تعمل بشكل صحيح.

## 🏠 اختبار الصفحة الرئيسية

### التحميل والعرض
- [ ] الصفحة تحمل بسرعة (أقل من 3 ثوان)
- [ ] جميع العناصر تظهر بشكل صحيح
- [ ] الألوان صحيحة (أخضر)
- [ ] الخطوط عربية واضحة
- [ ] الصور تحمل بشكل صحيح

### الرأس (Header)
- [ ] الشعار يظهر بشكل صحيح
- [ ] زر الإشعارات يظهر
- [ ] زر الإدارة (⚙️) يظهر
- [ ] الرأس ثابت عند التمرير

### البحث
- [ ] مربع البحث يستقبل النص
- [ ] البحث يعمل في الوقت الفعلي
- [ ] النتائج تتغير أثناء الكتابة
- [ ] البحث يعمل مع الكلمات العربية
- [ ] البحث يعمل مع الوسوم

### التصفية
- [ ] زر "الكل" يعرض جميع الأعمال
- [ ] زر "مطعم" يعرض المطاعم فقط
- [ ] زر "خدمات" يعرض الخدمات فقط
- [ ] زر "متجر" يعرض المتاجر فقط
- [ ] زر "جمعية" يعرض الجمعيات فقط
- [ ] الأزرار النشطة تتغير اللون

### الترتيب
- [ ] قائمة الترتيب تعمل
- [ ] "الأحدث" يرتب حسب التاريخ
- [ ] "اسم العمل" يرتب أبجدياً
- [ ] الترتيب يعمل مع التصفية

### بطاقات الأعمال
- [ ] الصور تظهر بشكل صحيح
- [ ] الأسماء واضحة
- [ ] الأوصاف مرئية
- [ ] الفئات تظهر بألوان مختلفة
- [ ] أزرار الواتساب والتفاصيل موجودة

### نافذة التفاصيل (Modal)
- [ ] النافذة تفتح عند الضغط على البطاقة
- [ ] الصورة الكبيرة تظهر
- [ ] المعلومات الكاملة تظهر
- [ ] الوسوم تظهر بشكل صحيح
- [ ] زر الواتساب يعمل
- [ ] زر الموقع يظهر (إن وجد)
- [ ] زر الإغلاق يعمل
- [ ] الضغط خارج النافذة يغلقها

### الإشعارات
- [ ] زر الإشعارات يظهر
- [ ] العداد يظهر (إن وجدت إشعارات جديدة)
- [ ] الضغط على الزر يعرض الإشعار
- [ ] نص الإشعار يظهر بشكل صحيح
- [ ] التاريخ يظهر بصيغة عربية
- [ ] الإشعار يغلق تلقائياً بعد 8 ثوان
- [ ] زر الإغلاق يعمل

### التذييل (Footer)
- [ ] معلومات البلدية تظهر
- [ ] اسم المطور يظهر
- [ ] رابط الواتساب يعمل
- [ ] رابط الإنستغرام يعمل
- [ ] حقوق النشر تظهر

### التوافق
- [ ] الموقع يعمل على Chrome
- [ ] الموقع يعمل على Firefox
- [ ] الموقع يعمل على Edge
- [ ] الموقع يعمل على Safari
- [ ] الموقع متجاوب على الهاتف
- [ ] الموقع متجاوب على التابلت
- [ ] الموقع متجاوب على الحاسوب

## ⚙️ اختبار لوحة الإدارة

### الدخول
- [ ] الضغط على ⚙️ يفتح لوحة الإدارة
- [ ] الرابط يعمل من الصفحة الرئيسية
- [ ] الرابط "العودة" يعود للصفحة الرئيسية

### التبويبات
- [ ] تبويب "إدارة الأعمال" يعمل
- [ ] تبويب "إدارة الإشعارات" يعمل
- [ ] تبويب "استيراد/تصدير" يعمل
- [ ] تبويب "الإعدادات" يعمل
- [ ] التبويب النشط يتغير لونه

- [ ] زر "حفظ العمل" يعمل
- [ ] رسالة النجاح تظهر

- [ ] العمل يظهر في الصفحة الرئيسية

#### تعديل عمل
- [ ] زر "تعديل" يملأ النموذج
- [ ] البيانات الموجودة تظهر بشكل صحيح
- [ ] يمكن تغيير البيانات
- [ ] زر "حفظ العمل" يحدّث البيانات
- [ ] التغييرات تظهر في القائمة والصفحة الرئيسية

#### حذف عمل
- [ ] زر "حذف" يطلب تأكيد
- [ ] الحذف يزيل العمل من القائمة
- [ ] الحذف يزيل العمل من الصفحة الرئيسية
