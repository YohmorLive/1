# 🚀 دليل النشر على GitHub Pages

هذا الدليل يشرح كيفية نشر Yohmor Live على الإنترنت بشكل مجاني باستخدام GitHub Pages.

## 📋 المتطلبات

- حساب بريد إلكتروني (لإنشاء حساب GitHub)
- متصفح ويب
- الملفات الكاملة للمشروع

## 🎯 الخطوات

### الخطوة 1: إنشاء حساب GitHub

1. انتقل إلى [github.com](https://github.com)
2. انقر على **Sign up** (في الزاوية العلوية اليمنى)
3. أدخل:
   - **Username:** اختر اسم مستخدم (مثل: `your-name`)
   - **Email:** بريدك الإلكتروني
   - **Password:** كلمة مرور قوية
4. انقر على **Create account**
5. أكمل التحقق من البريد الإلكتروني

### الخطوة 2: إنشاء مستودع جديد

1. بعد تسجيل الدخول، انقر على **+** في الزاوية العلوية اليمنى
2. اختر **New repository**
3. ملء النموذج:
   - **Repository name:** `yohmor-live`
   - **Description:** `منصة الأعمال والخدمات في بلدة يحمر الشقيف`
   - **Public:** اختر (مهم جداً!)
   - **Initialize this repository with:** لا تختر شيء
4. انقر على **Create repository**

### الخطوة 3: رفع الملفات

#### الطريقة الأولى: الرفع المباشر (الأسهل)

1. في صفحة المستودع الجديد، انقر على **Add file** > **Upload files**
2. اسحب جميع الملفات من مجلدك إلى منطقة التحميل:
   - index.html
   - admin.html
   - styles.css
   - app.js
   - admin.js
   - data.json
   - README.md
   - QUICKSTART.md
   - PROJECT_SUMMARY.md
   - DEPLOYMENT_GUIDE.md
   - .gitignore

3. في أسفل الصفحة، انقر على **Commit changes**

#### الطريقة الثانية: استخدام Git (للمتقدمين)

إذا كان لديك Git مثبت:

```bash
# انسخ رابط المستودع من GitHub

# في مجلد المشروع:
git init
git add .
git commit -m "Initial commit - Yohmor Live v1.0"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/yohmor-live.git
git push -u origin main
```

### الخطوة 4: تفعيل GitHub Pages

1. في صفحة المستودع، انقر على **Settings** (الإعدادات)
2. من القائمة الجانبية، اختر **Pages**
3. تحت **Source**:
   - اختر **Deploy from a branch**
   - اختر **main** من القائمة الأولى
   - اختر **/ (root)** من القائمة الثانية
4. انقر على **Save**

### الخطوة 5: انتظر النشر

- GitHub سيبدأ في نشر موقعك تلقائياً
- انتظر 1-2 دقيقة
- ستظهر رسالة خضراء تقول "Your site is live at..."

### الخطوة 6: الوصول إلى موقعك

موقعك سيكون متاحاً على:
```
https://YOUR_USERNAME.github.io/yohmor-live
```

مثال:
```
https://molleik7.github.io/yohmor-live
```

## 🔄 تحديث الموقع

### بعد إجراء تغييرات:

#### الطريقة الأولى: الرفع المباشر
1. افتح صفحة المستودع
2. انقر على الملف الذي تريد تعديله
3. انقر على أيقونة القلم (Edit)
4. اجعل التغييرات
5. انقر على **Commit changes**


## 🆘 حل المشاكل

### المشكلة: الموقع لا يظهر
**الحل:**
- تأكد من أن المستودع عام (Public)
- تأكد من وجود `index.html` في الجذر
- انتظر 5 دقائق إضافية
- امسح ذاكرة التخزين المؤقت للمتصفح

### المشكلة: الصور لا تظهر
**الحل:**
- استخدم روابط مطلقة للصور
- تأكد من أن الروابط تعمل
- جرب صورة من الإنترنت

### المشكلة: الأنماط لا تظهر بشكل صحيح
**الحل:**
- تأكد من أن `styles.css` في نفس المجلد
- امسح ذاكرة التخزين المؤقت (Ctrl+Shift+Delete)
- حاول متصفح مختلف

### المشكلة: البيانات لا تُحفظ
**الحل:**
- هذا طبيعي! البيانات تُخزن محلياً فقط
- استخدم "تصدير" لحفظ البيانات
- استخدم "استيراض" لتحميل البيانات
